# Calorie Tracker

O aplicație simplă pentru a urmări caloriile zilnice consumate.

## Cum se folosește

1. Instalează Flask:
   ```bash
   pip install -r requirements.txt
   ```

2. Inițializează baza de date:
   ```bash
   python init_db.py
   ```

3. Pornește aplicația:
   ```bash
   python app.py
   ```

4. Accesează în browser:
   ```
   http://localhost:5000
   ```
